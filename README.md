# Recruitment Task Manager App

## Overview

This is a task management application for the recruitment process, built using React Native and Firebase.

## Dependencies

- React (v18.2.0)
- React Native (v0.73.4)
- React Navigation or React Navigation Native (v6.1.10)
- Firebase or React Native Firebase (v10.8.0)
- React Native Vector Icons
- React Native Gesture Handler
- React Native Simple Radio Button (v2.7.4)
- @react-navigation/stack (v6.3.21)
- Firebase Database (Assuming part of the Firebase package)
- Firebase Authentication (Assuming part of the Firebase package)
- React Native Elements (v3.4.3)
- Axios (v0.25.0)
- React Native Firebase Analytics (Assuming part of the React Native Firebase package)
